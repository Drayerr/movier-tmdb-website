# Website made with [TMDB API](https://developer.themoviedb.org/reference/getting-started) data.
# Built with React + Vite.

## Environment
* To run the project you may get an Token from [TMDB API](https://developer.themoviedb.org/reference/getting-started)
* Create a .env.local file at the project root and fill the VITE_API_KEY="Baerer PASTE_YOUR_TOKEN"

## Dependencies
* 1 Run ```npm i``` to install the packages

## Running
* 1 Run ```npm run dev``` to run the project locally
